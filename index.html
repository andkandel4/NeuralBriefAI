// faq

Common questions.

Do I need to know how to code?
No. n8n is a visual drag-and-drop tool. You import the file, paste in your API keys, and activate. The setup guide explains every single step in plain English.
Does it cost money to run?
n8n has a free self-hosted version. The Claude API costs roughly $0.001 per summary — running daily that's less than $0.50/month. You can also use n8n cloud for ~$20/mo if you don't want to self-host.
How long does setup actually take?
Most people are fully live in under 30 minutes. Importing the workflow is instant. Getting API keys takes the most time — the guide links you directly to every page you need.
Can I customize the topics or sources?
Yes. The workflow comes with AI/tech news pre-configured, but you can swap in any RSS feeds you want — finance, sports, crypto, whatever niche you're targeting.
What if I get stuck?
You get access to a Discord community included with your purchase. Drop a message and you'll get help fast.
